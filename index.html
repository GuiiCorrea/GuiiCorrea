16. Explique a diferença entre front-end e back-end.

Front-end é a parte do site ou aplicação visível ao usuário, responsável pelo design, layout e interação. Back-end é a parte que roda no servidor, lidando com banco de dados, lógica de negócio e processamento de dados.

17. Descreva a função de cada uma das seguintes tecnologias: HTML, CSS e JavaScript.

HTML: estrutura o conteúdo da página.
CSS: estiliza a aparência e layout dos elementos.
JavaScript: adiciona interatividade e lógica à página.

18. O que é responsividade e por que ela é importante no desenvolvimento web?

Responsividade é a capacidade de um site se adaptar a diferentes tamanhos de tela e dispositivos. É importante para melhorar a experiência do usuário e garantir que o site funcione bem em celulares, tablets e desktops.

19. Explique o que é uma aplicação web moderna.

Uma aplicação web moderna utiliza tecnologias atuais, como HTML5, CSS3, JavaScript (e frameworks), comunicação via APIs, e pode ser responsiva, interativa e segura, funcionando em diferentes dispositivos e navegadores.

20. Cite duas boas práticas no desenvolvimento web e explique sua importância.

1) Manter código limpo e organizado: facilita manutenção e colaboração.
2) Garantir responsividade: melhora a experiência do usuário em diversos dispositivos.
